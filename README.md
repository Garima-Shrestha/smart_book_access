# Novella

Novella is a digital book rental mobile application built with Flutter that
allows users to browse, rent, and read books at affordable prices. Rentals
last between one and thirty days, with payments processed through Khalti.
Books are accessible through a built-in PDF reader that supports offline
reading, bookmarking, quote saving, and progress tracking. The application
also makes use of device sensors, using the accelerometer to turn pages and
the proximity sensor to open the bookmark list.
